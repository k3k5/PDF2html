# PDF2html
Personal wrapper for pdf2htmlEX

## Prerequisites
Docker
NodeJS

## Usage
### Download Code
```sh
git clone git@github.com:k3k5/PDF2html.git
cd PDF2html
chmod +x runner.sh
```

### Running
* Copy the PDF File into the PDF2html directory
* Run the code:
```sh
./runner.sh file_to_convert.pdf result_file.html
```
